# 🚀 Build & Deployment Guide

## Build Instructions

### Android Build
```bash
flutter build apk --release
flutter build appbundle --release
```

### iOS Build
```bash
flutter build ios --release
flutter build ipa --release
```

### Web Build
```bash
flutter build web --release
```
