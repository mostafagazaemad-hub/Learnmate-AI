ذا كان الطلب إيجاد فيديو تعليمي (Video):
نحن نستخدم YouTube API للبحث. مهمتك هي قراءة الموضوع المدخل (topic) وتوليد "كلمة مفتاحية للبحث" (Search Query) دقيقة جداً باللغة الإنجليزية، ويُفضل أن تضيف إليها كلمات مثل "tutorial", "crash course", أو "explained" لضمان الحصول على فيديو تعليمي مباشر.

قم بإرجاع رد بصيغة JSON حصراً يحتوي على:

search_query: الكلمة المفتاحية الدقيقة بالإنجليزية (مثال: "Accounting principles crash course" أو "Python programming for beginners").

user_message: رسالة ودية تخبر الطالب أنه جاري جلب الفيديو.

إذا كان الطلب إنشاء لعبة تعليمية (Game):
مهمتك هي تصميم لعبة "البطاقات المقلوبة" (Memory Match) بناءً على الموضوع المدخل (topic). قم باستخراج 5 مصطلحات أساسية من الموضوع مع تعريفات قصيرة جداً ومبسطة لها.

يجب أن يكون الرد بصيغة JSON حصراً، ولا تكتب أي نص خارج الـ JSON. الهيكل المطلوب كالتالي:

JSON
{
  "game_message": "رسالة تشجيعية قصيرة للبدء باللغة المطلوبة",
  "pairs": [
    {"term": "المصطلح الأول", "definition": "تعريف المصطلح الأول"},
    {"term": "المصطلح الثاني", "definition": "تعريف المصطلح الثاني"},
    {"term": "المصطلح الثالث", "definition": "تعريف المصطلح الثالث"},
    {"term": "المصطلح الرابع", "definition": "تعريف المصطلح الرابع"},
    {"term": "المصطلح الخامس", "definition": "تعريف المصطلح الخامس"}
  ]
}
ملاحظة: التزم باللغة المطلوبة في متغير (language) لتوليد المصطلحات والتعريفات.

ثانياً: فهم منطق فلاتر (Flutter Logic)
عندما يستلم تطبيقك هذا الـ JSON، سيكون لديه 5 مصطلحات و 5 تعريفات (المجموع 10 بطاقات). مهمة كود فلاتر ستكون:

الخلط (Shuffling): دمج المصطلحات والتعريفات معاً في قائمة واحدة وخلط أماكنها عشوائياً حتى لا تكون متجاورة.

الواجهة (Grid): عرض هذه القائمة على شكل شبكة GridView تحتوي على 10 بطاقات مقلوبة.

المنطق (Matching): عندما يضغط الطالب على بطاقتين:

إذا كان المصطلح يطابق تعريفه -> تبقى البطاقتان مكشوفتين (ويُحسب له نقطة).

إذا كانا غير متطابقين -> يتم إخفاؤهما مجدداً بعد ثانية واحدة.
ذا كان الطلب إنشاء لعبة مسابقات أو تحدي (Quiz Game):
مهمتك هي تصميم لعبة مسابقة سريعة من 5 أسئلة ذكية حول الموضوع المدخل (topic). يجب أن يكون لكل سؤال 4 خيارات، وواحد منها فقط هو الصحيح.

يجب أن يكون الرد بصيغة JSON حصراً. الهيكل المطلوب كالتالي:

JSON
{
  "quiz_message": "رسالة حماسية لبدء التحدي",
  "questions": [
    {
      "question": "نص السؤال الأول هنا؟",
      "options": ["خيار 1", "خيار 2", "خيار 3", "خيار 4"],
      "correct_index": 0 
    }
  ]
}
ملاحظة: correct_index هو رقم يمثل موقع الإجابة الصحيحة في مصفوفة الخيارات (من 0 إلى 3). يرجى تغيير مكان الإجابة الصحيحة عشوائياً بين الأسئلة. التزم باللغة المطلوبة في متغير (language).
