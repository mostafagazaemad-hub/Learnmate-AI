# 🎓 Learnmate AI

> تطبيق ذكي يجمع بين تكنولوجيا الذكاء الاصطناعي والتعليم الحديث

## 📱 عن المشروع

**Learnmate AI** هو تطبيق موبايل متطور يحتوي على **12 أداة مدمجة بالذكاء الاصطناعي** تساعد الطلاب على تعلم أي مساق أو موضوع تعليمي بطرق مختلفة وفعّالة.

يوفر التطبيق:
- 🤖 أدوات ذكية مدعومة بـ Google Generative AI
- 📚 تجربة تعليمية شخصية ومخصصة
- 🔐 مصادقة آمنة عبر Firebase
- 💾 تخزين سحابي للبيانات والملفات
- 🎨 واجهة مستخدم حديثة وسهلة الاستخدام

---

## 🛠️ التقنيات المستخدمة

| التقنية | الوصف |
|---------|-------|
| **Flutter** | إطار عمل لتطوير تطبيقات متعددة المنصات |
| **Dart** | لغة البرمجة الأساسية |
| **Firebase** | للمصادقة والتخزين السحابي وقاعدة البيانات |
| **Google Generative AI** | لتطبيق خوارزميات الذكاء الاصطناعي |
| **Google Sign-In** | للمصادقة الآمنة |
| **Google Fonts** | لتصميم احترافي للخطوط |
| **Image Picker** | لاختيار الصور من الجهاز |
| **File Picker** | لاختيار الملفات |
| **Flutter TTS** | لتحويل النص إلى كلام |
| **Shimmer** | لتأثيرات تحميل جميلة |
| **WebView Flutter** | لعرض محتوى الويب |
| **Video Player** | لتشغيل الفيديوهات |

## 🚀 خطوات التشغيل

### 1. استنساخ المستودع
```bash
git clone https://github.com/mostafagazaemad-hub/Learnmate-AI.git
cd Learnmate-AI
```

### 2. تثبيت المتعلقات
```bash
flutter pub get
```

### 3. إضافة مفاتيح Firebase و Google API
- قم بإنشاء مشروع Firebase من [Firebase Console](https://console.firebase.google.com)
- أضف ملفات الإعدادات المطلوبة

### 4. تشغيل التطبيق
```bash
flutter run
```
